# Study Material

이 저장소는 공부한 알고리즘 문제 풀이와 학습 자료를 정리하는 공간입니다.

## 📚 구성

- `algorithm/` - 알고리즘 문제 풀이 코드

## ✨ 현재 포함된 자료

- `algorithm/codeforce_A - Theatre Square.cpp`
- `algorithm/codeforce_A - Watermelon.cpp`
- `algorithm/codeforce_A - Way Too Long Words.cpp`

## 🛠️ 사용 계획

- 알고리즘 문제 풀이를 주제별로 정리
- 새로운 문제를 풀 때마다 풀이와 메모 추가
- 자주 참조하는 개념이나 자료를 별도 문서로 정리

## 🚀 추천 추가 항목

- `README`에 공부한 알고리즘/자료 목록 추가
- `algorithm/` 안에 `README.md`를 만들고 분류별로 정리
- `notes/` 또는 `docs/` 폴더를 만들어 학습 노트 저장

## 📌 앞으로 하기 좋은 것

1. `algorithm/` 내부에 `greedy/`, `bruteforce/`, `dp/` 같은 폴더 추가
2. 문제별 풀이 요약과 시간 복잡도 기록
3. 풀이에 참고한 링크나 개념 정리 추가
